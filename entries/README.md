# Entries

There’s a lot to see and explore here, but don’t worry — this guide will walk you through everything step by step and help you understand how it all works. &#x20;

## English/Irish website selection&#x20;

At the top left of the screen, you will see an SETU dropdown button which allows you choose between if you would like to edit the English version of the website or the Irish section of the website.&#x20;
