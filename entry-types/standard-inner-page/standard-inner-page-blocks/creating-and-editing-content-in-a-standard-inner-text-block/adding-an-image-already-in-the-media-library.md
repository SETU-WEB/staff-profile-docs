# Adding an image - already in the media library

&#x20;Click image icon, this will open the media library window. From here you can choose directory and search for an image that is has already been uploaded using the search box. If you don't know the image name you can set the **view as** thumbnail (middle icon next to search bar) were you can see what the image looks like .You can also set how many images you want to show to allow you view a bigger group of images . There is also a view option which allows you to filter the image search for different options.&#x20;
